Portfolio Website


  • Website: Live Site | Github URL: Project Link

  • This is a fully responsive personal portfolio built with Next.js, Tailwind CSS, and Framer Motion. It features sections for projects, experience, and contact, with smooth UI transitions. Deployed on Google App Engine with a custom domain and automated CI/CD pipeline for zero-downtime deployments.


  • Tech Stack
    • Frontend: Next.js, Tailwind CSS, Framer Motion
    • Deployment: Google App Engine, Google Cloud Build
    • CI/CD: GitHub Actions, cloudbuild.yaml
  • Features
    • Responsive design with smooth transitions
    • Component-based architecture for scalability
    • Automated builds and deployments
    • Custom domain and zero-downtime updates
