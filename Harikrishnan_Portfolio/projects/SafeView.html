SafeView


  • Github URL: Project Link

  • SafeView is a real-time, privacy-focused Chrome extension for NSFW content moderation using TensorFlow.js and a fine-tuned MobileNetV3-Small model. It features in-browser image classification, dynamic content obfuscation, and user-configurable filtering settings.


  • Tech Stack
    • Frontend: JavaScript, HTML, CSS
    • ML: TensorFlow.js, MobileNetV3-Small
    • Browser: Chrome Extension APIs
  • Features
    • Real-time NSFW content moderation
    • Dynamic content obfuscation
    • User-configurable filtering
    • Optimized frame-processing for high FPS video
